    비즈니스 캔버스

    초심이자, 길이 보이지 않을 때 다시 돌아올 공간

    사업이 잘될지 안될지 보입니다.

    • Problem : 고객이 안고 있는 과제
    • Solution : 어떤 기능으로 사용자의 과제를 해결할 것인지
    • Key Metrics : 솔루션을 제공하여서 발생하는 측정가능한 지표
    • Value Proposition : 프로덕트가 제안하는 가치
    • High-Level Concept : 한마디로 전달될 수 있는 문구
    • Unfair Advantage : 자사만이 제공 가능한 솔루션
    • Channel : 고객군에 접근하기 위한 방안
    • Segments : 타깃 사용자
    • Early Adopters : 가장 빨리 사용할 고객군
    • Cost Structure : 프로덕트 실현 위한 초기 투자 비용 및 운영 비용
    • Revenue Streams : 프로덕트 실현 통한 수익과 수익원

    4.여자한의사가 진료하는곳이 별로 없다.

    한의원 CASE 3

    Problem Solution Value Proposition Unfair Advantage Segments
    1. 체중 : 다이어트 방식은 많다. 하지만 고객은 확실한 방법을 모르고 끝까지 해낼 의지가 없다.
    2. 당뇨 : 당뇨약을 먹어도 당화혈색소가 그대로인 환자가 많다.
    3. 만성피로, 만성염증 : 특별한 일 없이 몸이 좋지 않다.

    Existing Alternatives
    1. 체중 : 기존 다이어트 한약, 양약, 다이어트 프로그램
    2. 당뇨 : 양약
    3. 만성피로, 만성염증 : 스테로이드, 각종 건기식, 한약    		 1. 한약
    2. 침치료
    3. 상담 및 생활습관 개선
    4. 연속혈당측정기 및 혈액검사    		 -수치로 확인할 수 있는 결과
    -따라오기만 하면 만들어지는 건강
    -의지가 없어도 따라가게 되는 프로그램    		 -함께 하는 다이어트팀
    -원장의 경험과 데이터, 의지
    -조직의 환자 장악력    		 -과체중
    -당뇨, 당뇨전단계
    -만성피로, 만성염증

    던지는 질문

    • 처음에 만들 팬을 정합니다. 누구인가요?
    • 고객의 문제를 풀면, 고객은 만족합니다. 문제는 무엇인가요?
    • 남들은 제공 못하는 나만 줄 수 있는 이득은 무엇인가요?
    • 나/상대의 강점, 약점, 기회, 위협은 무엇인가요?
    • 그래서 만들어질 결과는 무엇인가요?
    • 매출을 확보하고 이익을 낼 수 있나요?
